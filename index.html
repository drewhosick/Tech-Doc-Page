Overview

Today we are going to look at some tags in HTML. You can use the navbar above to go to them or scroll down.

Paragraphs

The Paragraph is part of HTML. The tag looks like this <p>.

It's the simplest form of HTML and is used to write text in a paragraph for the reader or user.

<p>This is a code example</p>

Headlines

Headlines are also important and denote what things below them will be about. For example, you might use a headline as a title to a page or to denote a section below it that will explain to the user, more about what you are talking about in your headline by using a paragraph.

Headlines can be created in HTML by using <hx> with x being replaced with 1 through 6 depending on the importance of the headline. There should only be one headline with an <h1>

<h3>This is a code example</h3>

Breaklines

Breaklines are used on elements that don't create a new line after themselves. While the end of a paragraph like so </p> will cause the browser to render a new line for the next element. Other elements are known as inline elements like links and images and you need to send a new line command or use CSS to make sure content after it will be on a new line.

They are represented with <br>. It's important to remember that you do not need </br> for this tag.

This is a code example<br>
This was in the same code snippet.

Styles

Styles are what give lines different looks. They can be applied inside tags like <p>, <h1> or most all other elements. Or they can be applied to items through a CSS sheet or Cascading Style Sheet.

When applying styles inline, you must use style="style goes here". You can also apply them with <style> and </style> tags in the head of the document.

Styles would look something like this line here:

style="font-size:16px;"

Javascript

Javascript is what makes a page come to life. It allows the programmer to give the user more than just a visual experience. You can use coding to do things like change items on the screen based on input from the user in an input box. Or it could be something as simple as doing math behind the scenes.

Javascript is included using the <script> tags and by either writing that scripting inline or by including it on a .js file through a link in the script tag to the file.

Code could look something like this:

const main-section = document.queryselector(.main-section);
